Crop Disease Detection System

An intelligent system developed to detect crop diseases and provide useful recommendations for farmers.

Features:
- Image-based crop disease detection
- Farmer advisory system with suggested solutions
- Crop price insights (if applicable)
- Easy-to-use interface for better accessibility

Tech Stack:
HTML, CSS, JavaScript (and basic image processing concepts)

This project aims to support farmers by combining technology with agriculture to improve productivity and decision-making.
